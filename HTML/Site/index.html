Conceitos do HTML

1. O que é HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

2. Tags

São usadas para estruturar o conteúdo de um documento, para formatar o texto, para adicionar imagens e outros elementos, etc. As tags HTML básicas são essenciais para criar qualquer documento HTML. Sem elas, um documento HTML não teria nenhum conteúdo e não seria exibido corretamente.

  • h1 até h6 : É usado para exibir o texto como título, quanto maior o número menor o tamanho
  • p : Formata o texto como um paragráfo
  • mark : Destaca o texto
  • small : Adiciona um texto menor do que o padrão
  • i : Deixa o texto em Italico
  • u : Sublinha o texto
  • strong : Deixa o texto em negrito
  • ol : Cria uma lista ordenada
  • ul : Cria uma lista não ordenada
  • li : Adiciona um elemento numa lista
  • a : Define um hyperlink, usado para ligar uma página a outra
  • hr : Cria uma quebra preenchida com uma linha
  • sub : Subscreve o texto
  • sup : Sobrescreve o texto
  • blockquote : Usado para fazer citações ou anotações.

3. Atributos das Tags

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são modificadores de um tipo de elemento HTML.

Por exemplo, podem ser aplicados na tag de hyperlink (<a>) os atributos href, para onde o usuário vai ser direcionado ao clicar no hyperlink, e target, em qual página a ação vai ocorrer. Numa tag de imagem (<img>) pode ser usado o atributo src, que indica a origem da imagem.
